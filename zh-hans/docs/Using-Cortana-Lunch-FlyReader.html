# 在Cortana中使用越飞阅读

本软件与Cortana(小娜)深度集成，你可以使用以下的命令借助Cortana来使用本软件。

# 支持命令列表

越飞阅读 开始阅读

越飞阅读 阅读 {某小说}

越飞阅读 打开 {某小说} 的章节列表

提示

如果你自定义了越飞阅读的称呼，则将命令中的 越飞阅读 换成 自定义的软件称呼 ，即可操作越飞阅读。

# 自定义Cortana称呼

你可以在 设置 >> 语音 中自定义 Cortana 对越飞阅读的称呼。

提示

在自定义称呼后，需要重启软件才能生效。
